# To Run : node Api.js


# Dependencies:
`

1) npm init -y
2) npm express nodemon compilex body-parser
3) also install the tailwind css  framework by running "npx tailwindcss init" in terminal and then installing it via package.
4) use bootstrap icons (<https://icons.getbootstrap.com/icons/play-fill/>) for the play button icon</
5) use JS Library ->  CodeMirror ()
6) add language from :-  <https://github.com/scriptnull/compilex>   -: for compiling code in real time
`
